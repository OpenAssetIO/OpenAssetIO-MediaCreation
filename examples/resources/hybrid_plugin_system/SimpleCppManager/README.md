# Simple C++ Manager

This directory is where the hybrid_plugin_system.ipynb notebook will
look for the SimpleCppManager plugin.

In order for that notebook to run, the SimpleCppManager plugin must be
built and installed here.

See the [SimpleCppManager README](https://github.com/OpenAssetIO/OpenAssetIO/blob/main/examples/manager/SimpleCppManager/README.md)
for more details.
